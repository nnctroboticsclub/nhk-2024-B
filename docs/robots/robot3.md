# Robot 3

## Pinmap

```
FEP
  INI: PC8
  RST: PC9
  (TX, RX): PC6 PC7

  |  F |  R
--+----+----
M4|PB10| PB2
M3| PB9| PB8
M2|PA11|PA10
M1| PA9| PA8
```

## コントローラー

|部品|機能|種類|
|:-:|:-:|:-:|
|足|前後|ジョイスティック|
|足|旋回|ジョイスティック|
|アーム|開閉|ボタン x2|
|アーム|昇降|ボタン x2|
